# Online Retail Store | Giftware Store
## Customer Cohort Analysis 
This Power BI project explores how wholesale clients shop at the online giftware store by tracking their long-term buying habits. The analysis uses weekly and monthly customer cohorts to show exactly how many shoppers are staying loyal versus those who stop buying.

## Process
- Data Collection Stage: Imported the excel (xlsx) file in power BI data model.
- Data Preparation stage: Cleaned the data by handling missing values, removing duplicates, select relevant data types etc.
- Data Modeling Stage: Built star schema model where sales-table represent "fact table" and oters are dimentional tables.
- Exploratory Data Analysis (EDA):Prepared several dashboards pages
